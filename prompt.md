# Projeto: Jogo 2D de Plataforma com Física

## Objetivo
Criar um jogo 2D de plataforma simples usando JavaScript, HTML e CSS com física realista.

## Requisitos
- Personagem que pode andar e pular
- Física de gravidade e colisão
- Plataformas para o personagem andar
- Controles responsivos (setas do teclado)
- Canvas HTML5 para renderização
- Código limpo e bem estruturado

## Funcionalidades Desejadas
1. Movimento horizontal (esquerda/direita)
2. Pulo com física realista
3. Colisão com plataformas
4. Gravidade constante
5. Visual simples mas funcional

## Critérios de Sucesso
- Jogo jogável sem bugs
- Física funciona corretamente
- Controles são responsivos
- Código é legível e mantível
